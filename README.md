# VertexComponentTable
MayaLT用統合コンポーネントエディタ
(MayaLT2016 Extension3 Shift-Jis)  
ソースに更新履歴は書いてあります。

現状は以下の機能に対応しています  
・頂点座標  
・頂点カラー  
・頂点法線  
・スキンウェイト  
要望あればIsuueにどうぞ
